Sons para Crescer

Um Website construído para uma clínica de Terapia da Fala.

Detalhes do Projecto

Para este projecto tinha de criar um Website para uma clínica de Terapia da Fala, cujo foco era providenciar informação básica sobre diferentes patologias, bem como as diferentes estratégias para trabalhar com essas mesmas patologias

É um site extremamente simples na sua essência, mas o cliente queria que o utilizador se focasse na informação que é dada, com o mínimo de distrações possíveis. (Ainda é um trabalho em progresso)

  • Cliente: SonsparaCrescer
  • Tarefas: Desenhar e Desenvolver um Website usando HTML, CSS e Javascript
  • Website: SonsparaCrescer
